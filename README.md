\# Plainsight



\*\*Category:\*\* Web / Recon  

\*\*Difficulty:\*\* Very Easy  

\*\*Author:\*\* Your Name



\## Description

A beginner web challenge. Open the page and find the hidden flag.



\## How to play

1\. Visit the challenge URL.

2\. Right-click → View Page Source (or press Ctrl+U).

3\. The flag is hidden in an HTML comment.



\## Hints

\- Hint 1: Try viewing the page source.

\- Hint 2: The flag uses the CYS{...} format.
